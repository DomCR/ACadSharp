Remarks

This static class includes constants for frequently used mathematical values and conversion factors between degrees, radians, and gradians. It also offers utility methods for comparing floating-point numbers with a configurable tolerance, normalizing angles, and rounding values. The methods are designed to address common issues with floating-point precision and angle normalization in mathematical computations.

Returns a new vector in which each component of the input vector is replaced with zero if its absolute value is less than the specified threshold.

public static T FixZero<T>(T vector, double threshold) where T : IVector, new()

Parameters

The input vector whose components are to be processed.

threshold double

The threshold below which a component is considered to be zero. Components with an absolute value less than this value are set to zero.

Returns

T

A new vector of type T with components set to zero if their absolute value is less than the specified threshold; otherwise, the original component value is retained.

Type Parameters

The type of vector. Must implement the IVector interface and have a parameterless constructor.

Remarks

This method does not modify the input vector. The returned vector is a new instance of type T. Use this method to eliminate near-zero noise in vector data.
