StoredValue

Gets or sets the value associated with the current group code, using the appropriate type based on the group code definition.

public object StoredValue { get; set; }

Property Value

object

Remarks

The type of the value must match the expected type for the group code as determined by the DXF specification. Assigning a value of an incorrect type will result in an exception. Supported types include string, numeric types, boolean, byte arrays, and custom handled objects, depending on the group code. When setting this property, the value is automatically converted or validated according to the group code's requirements.
