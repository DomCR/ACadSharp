TryAdd(string, ExtendedData)

Adds the specified extended data for the given application name if it does not already exist, and returns the stored extended data.

public ExtendedData TryAdd(string appName, ExtendedData extendedData)

Parameters

appName string

The name of the application to associate with the extended data. Cannot be null.

extendedData ExtendedData

The extended data to add if no data is already associated with the specified application name. Cannot be null.

Returns

ExtendedData

The existing extended data associated with the specified application name if it was already present; otherwise, the newly added extended data.

Remarks

If extended data for the specified application name already exists, this method does not overwrite it. Use this method to ensure that only one instance of extended data is associated with each application name.
