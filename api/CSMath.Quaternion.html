Returns

Remarks

The axis parameter should be a unit vector to ensure the resulting quaternion represents a valid rotation. The method assumes a right-handed coordinate system.

dest XYZ

The target direction vector to align with. Must be a non-zero vector.

Returns

Quaternion

A quaternion that rotates the origin vector to align with the destination vector.

Remarks

If the origin and destination vectors are opposite, the method selects an arbitrary orthogonal axis for the rotation. Both input vectors should be normalized for accurate results.
