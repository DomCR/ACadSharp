GetAngle(XYZ)

Calculates the angle, in radians, between this vector and the specified vector.

public double GetAngle(XYZ dir)

Parameters

dir XYZ

The vector to which the angle is measured.

Returns

double

The angle, in radians, between this vector and dir. The value is in the range [0, π].

Remarks

If either vector has zero length, the result may not be meaningful. The method returns 0 if the vectors are parallel and point in the same direction, and π if they are parallel and point in opposite directions.

GetAngle(XYZ, XYZ)

Calculates the signed angle, in radians, between this vector and the specified direction vector, using the given normal to determine the sign.

public double GetAngle(XYZ dir, XYZ normal)

Parameters

dir XYZ

The direction vector to which the angle is measured.

normal XYZ

The normal vector that defines the orientation of the plane in which the angle is measured. The sign of the angle is determined based on the direction of this normal.

Returns

double

The signed angle, in radians, between this vector and dir. The value is in the range [0, 2π].

Remarks

The returned angle is measured in the plane defined by this vector and dir, with the sign determined by the right-hand rule using normal. If the vectors are parallel, the angle is 0 or π depending on their direction.
