Splits the plain text into an array of lines based on common line break sequences.

public string[] GetPlainTextLines()

Returns

string[]

An array of strings, where each string represents a line of text. The array may contain empty strings if the plain text includes consecutive line break sequences.

Remarks

The method splits the text using the following line break sequences: carriage return and line feed ("\r\n"), carriage return ("\r"), line feed ("\n"), and the Unicode paragraph separator ("\P"). The resulting array includes all lines, including empty ones.
