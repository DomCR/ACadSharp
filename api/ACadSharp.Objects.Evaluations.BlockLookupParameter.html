Represents a BLOCKLOOKUPPARAMETER object, used in AutoCAD to drive block geometry from a lookup table in the Block Properties Table (the "Lookup" rows of the Customize tab of a dynamic block).

Remarks

Object name ObjectBlockLookupParameter
Dxf class name BlockLookupParameter

Minimal implementation: only the common block-element prefix (which carries Id, code 90 — the key found in ACAD_ENHANCEDBLOCKDATA) and the display name are decoded. The lookup table itself is not decoded.
