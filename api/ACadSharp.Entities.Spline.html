UpdateFromFitPoints(uint)

Updates the spline's control points, knots, and weights based on the current fit points and tangents.

public bool UpdateFromFitPoints(uint iterationLimit = 255)

Parameters

iterationLimit uint

The maximum number of iterations allowed for refining the tangents. Defaults to MaxValue.

Returns

bool

true if the spline was successfully updated; otherwise, false.

Remarks

This method performs the following operations:

  • Validates the spline's degree, fit points, and knot parametrization.
  • Generates knots and control points based on the fit points and tangents.
  • Refines the tangents iteratively if necessary, up to the specified iteration limit.
  • Assigns uniform weights to the control points.
The method returns false if the spline's degree is not 3, if there are fewer than two fit points, if the knot parametrization is set to Custom, or if tangent refinement fails.
