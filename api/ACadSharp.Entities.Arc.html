Arc(XYZ, XYZ, XYZ, XYZ)

Initializes a new instance of the Arc class that passes through the specified start and end points, is centered at the given point, and lies in the plane defined by the specified normal vector.

public Arc(XYZ center, XYZ start, XYZ end, XYZ normal)

Parameters

center XYZ

The center point of the arc. Defines the origin of the arc's circle.

start XYZ

The start point of the arc. Must lie on the circle defined by the center and radius.

end XYZ

The end point of the arc. Must lie on the circle defined by the center and radius.

normal XYZ

The normal vector defining the plane in which the arc lies. Must be a non-zero vector.

Remarks

The arc is constructed in the plane defined by the normal vector, with its center at the specified point. The start and end points determine the angular span of the arc. The direction from start to end is determined by the order of the points and the orientation of the normal vector.
