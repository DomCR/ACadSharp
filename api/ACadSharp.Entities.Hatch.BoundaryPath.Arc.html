public Arc(Circle circle)

Parameters

circle Circle

The circle or arc from which to construct the arc. Must not be null.

Remarks

If the provided circle is an arc, the arc's center, radius, start angle, and end angle are used. If the provided circle is a full circle, the arc will represent a full circle with a start angle of 0 and an end angle of 2π. The arc is always constructed in a counterclockwise direction.
