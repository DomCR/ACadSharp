Remarks

Use this class to obtain an appropriate ICadReader instance for supported CAD file types such as DWG and DXF. The factory selects the correct reader implementation based on the file extension. This class does not validate file existence or content; it only inspects the file extension.

Creates an ICadReader instance for the specified CAD file format.

public static ICadReader CreateReader(string filename, NotificationEventHandler notification = null)

Parameters

filename string

The path to the CAD file to read. The file extension determines the reader type. Must not be null or empty.

notification NotificationEventHandler

An optional event handler for receiving notifications during the reading process. Can be null if no notifications are needed.

Returns

ICadReader

An ICadReader instance capable of reading the specified CAD file.

Remarks

Supported file formats include DWG and DXF. The method selects the appropriate reader based on the file extension.

NotSupportedException

Thrown if the file extension is not supported.

GetFileFormat(string)

Determines the CAD file format based on the file extension of the specified filename.

public static CadFileFormat GetFileFormat(string filename)

Parameters

filename string

The path or name of the file whose format is to be identified. The file extension is used to determine the format.

Returns

CadFileFormat

A value of the CadFileFormat enumeration that corresponds to the file's extension. Returns DWG for ".dwg" files, DXF for ".dxf" files, or Unknown if the extension is not recognized.

Remarks

The method does not validate whether the file exists or whether the file content matches the extension. Only the file extension is considered when determining the format.
