GetLengthSquared<T>(T)

Calculates the sum of the squares of the vector's components.

public static double GetLengthSquared<T>(this T vector) where T : IVector

Parameters

vector T

The vector whose squared length is to be calculated. Must not be null.

Returns

double

The sum of the squares of all components of the vector. This value represents the squared length (magnitude) of the vector.

Type Parameters

T

The type of the vector, which must implement the IVector interface.

Remarks

This method does not compute the square root of the sum. To obtain the actual length (magnitude) of the vector, use Math.Sqrt on the returned value. This method is useful for performance-sensitive scenarios where comparing squared lengths is sufficient.
