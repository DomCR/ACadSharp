GetSystemValue(int, CadHeader)

Retrieves the value of a system variable from the specified CAD header based on the provided code and reference type.

public object GetSystemValue(int code, CadHeader header)

Parameters

code int

The code identifying the system variable to retrieve. The interpretation of this code depends on the reference type associated with the variable.

header CadHeader

The CAD header object from which to retrieve the system variable value. Cannot be null.

Returns

object

An object representing the value of the requested system variable. The type and meaning of the returned value depend on the reference type and the specified code.

Remarks

The method determines how to retrieve the value based on the reference type associated with the system variable. The returned value may be a direct property, a handle, a name, a count, or a raw value, depending on the context. Callers should cast the result to the expected type based on the variable being accessed.
